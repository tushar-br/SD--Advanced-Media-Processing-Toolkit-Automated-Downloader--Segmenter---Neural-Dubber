# Screenshots Directory

This folder contains UI screenshots for project documentation and reports.

## Required Screenshots

1. **Home Page** - Main interface with input field
2. **Processing Page** - Real-time progress and logs
3. **Results Page** - Download buttons and success message

## How to Capture Screenshots

1. Run the application using `RUN_PROJECT.bat`
2. Navigate through each stage of the workflow
3. Take screenshots using:
   - Windows: `Win + Shift + S`
   - Or use Snipping Tool
4. Save screenshots here with descriptive names:
   - `01_home_page.png`
   - `02_processing.png`
   - `03_results.png`

## Usage in Report

These screenshots should be included in:
- Project documentation
- Final semester report
- Presentation slides
- Viva demonstration materials
